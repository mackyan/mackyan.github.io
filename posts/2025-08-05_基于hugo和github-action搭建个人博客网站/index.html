基于Hugo和Github Action搭建blog

August 5, 2025 · 1 min

欢迎来到我的 Hugo 博客！#

  1. macOS安装Hugo
brew install hugo

验证安装

hugo version

实际安装版本：hugo v0.148.2

  1. 创建Hugo博客项目
hugo new site mackyan --format yaml
cd mackyan
  1. 添加主题

选择主题：以PaperMod为例

git init
git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

如果是 clone 了其他人的博客项目进行修改，则需要用以下命令进行初始化：

git submodule update –init –recursive

如果需要同步主题仓库的最新修改，需要运行以下命令：

git submodule update –remote

  1. 设置主题
vim hugo.yaml

添加内容

theme: [“PaperMod”]

  1. 写第一篇文章
hugo new posts/first.md

进入 content/posts/first.md，编辑内容如下：

---
title: "First Blog Post"
date: 2025-08-03
draft: false
---

欢迎来到我的 Hugo 博客！
  1. 本地预览博客
hugo server -D

打开浏览器访问：http://localhost:1313

  1. 配置博客信息（可选） 修改hugo.yaml中加入更多信息：
baseURL = "https://yourusername.github.io/"
title = "My Hugo Blog"
theme = "PaperMod"
languageCode = "zh-cn"
paginate = 10

[params]
  author = "Mack"
  ShowReadingTime = true
  ShowPostNavLinks = true
